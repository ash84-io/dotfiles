# dotfiles

> Shell is faster than hand 🤘

### use 

- SYSTEM
```shell 
> make brew
> make zsh 
> make python
> make rust
> make home 
```

- IDE 
```shell
> make settings
```

### structure
```
.

├── python.sh
├── rust.sh
└── zsh.sh
```
